Domplate 文档

用法

基本用法

要在页面上应用 domplate，需要在 <head> 标签内引用 domplate.js，并使用 domplate 提供的 api 对指定的 element 元素进行初始化：

<html>
<head>
  <script src="‌path/to/domplate.js‍"></script>
</head>
<body>
  <div id="container">
    <p>Hello, {{ name }}!</p>
  </div>
  <script>
    ‌domplate( "#container" )‍.load( { name: "domplate" } );
  </script>
</body>
</html>

domplate 将以特定的模板语法解析 #container 元素，并根据数据进行首次渲染。

脚本文件下载地址

也欢迎 fork domplate 的仓库：https://github.com/ChineseDron/domplate

使用模块加载器

如果页面上有 commonjs 模块加载器（如 require.js），也可以使用模块加载器的 use 方法载入 domplate.js，或者用 require 函数或它打包在一起。

其它说明

一个 domplate(...) 调用，称为一个 domplate 应用。

同一个页面上允许多个 domplate 应用，但原则上应用之间的容器元素不可以嵌套，如果存在嵌套情况，则有以下两种可能：

  1. 先初始化父容器应用，后初始化子孙容器应用，则子孙容器应用无效；
  2. 先初始化子容器应用，后初始化祖父容器应用，则两个应用均可独立运作，但是父应用在遇到子容器应用所处的 dom 元素在处理时会被忽略；
