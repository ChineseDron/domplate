ctrl.methods() 方法

ctrl.‌methods‍( { ‌methodA‍: fn, ‌methodB‍: fn, ... } );

methods() 用来创建控制器的方法，它只接受一个 key-value 形式的对象参数，key 用来作为方法名称，value 作为方法函数，这些函数将被内部的包装器处理，用 methods() 创建方法之后，可直接在控制器 ctrl 上访问这些方法，如 ctrl.methodA();

  1. 调用这些控制器方法时，会自动更新 HTML 视图；
  2. 调用这些控制器方法，返回 ctrl 对象；
