Domplate 文档

domplate() 函数

var ctrl = ‌domplate‍( element );`

将页面中已存在的 element 元素作为 dompalte 模板初始化，并返回一个控制器对象 ctrl。

element 参数可以是一个 dom 对象的引用，也可以是一个字符串，如果是一个字符串，domplate 内部以 document.querySelector( element ) 将其转化成一个 dom 对象。

调用 dompalte 函数时，将对指定的 dom 元素进行解析，如果该 dom 元素以及其任何父级 dom 元素未被 domplate 解析过，则初始化一个控制器对象，并将该控制器对象返回，上述的示例代码中，ctrl 就是一个控制器对象，可以通过 ctrl 来控制 element 视图。

初始化时传入的 element 对象，称为 ctrl 控制器的根节点。

而如果调用 domplate 函数时传入的 element 已经是某个控制器的根节点，返直接返回所在的控制器；如果 element 不是根节点，但它的其中一个父节点是根节点时，返回 null。

一个页面上允许同时存在多个 domplate 应用，这点在 用法 一节中已有说明。
