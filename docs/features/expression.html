Domplate 文档

表达式

以 {{ 和 }} 这组符号围起来的部分称为 domplate 表达式，domplate 表达式其实就是标准的 JS 表达式，domplate 将表达式的计算结果打印在表达式所在的位置。

在文本中应用表达式

<a href="http://ucren.com">‌{{ blogName }}‍</a>

在 HTML 属性中应用表达式

HTML 属性中，可直接在其值部分以表达式的形式填充：

<p class="box ‌{{ skinBox }}‍" style="color: ‌{{ color }}‍;" custom="‌{{ custom }}‍">test</p>

在 IE8 中，因为使用了 {{ 和 }} 这样的语法，有些属性在浏览器 HTML 自然解析时会失败：

  • style 属性，由于语法不符合 css，整个 style 属性会被 IE8 移除
  • on... 等事件属性，如果造成语法不符合 js，on... 属性不会被 IE8 移除，但是会报错
  • checked,selected... 等等具有 boolean 意义的属性，导致属性不可用

如果您考虑 IE8 的兼容，在表达式应用在上述属性时，应该对属性名称加上 d- 前缀：

<p ‌d-style‍="color: {{ color }};" ‌d-onclick‍="{{ scriptCode }};">test</p>

当然，如果不在乎 IE8，可以不用加 d- 前缀。

使用转义符 \ 可避免表达式解析

若希望在页面打印 { 和 } 的原文，需要前导转义符 \：

<p>domplate 支持表达式 ‌\{\{‍expression‌\}\}‍</p>
