事件属性

表达式 也可以直接应用在事件当中：

<button onclick="alert('‌{{ fruit.name }}‍');">Show</button>

为了避免事件在浏览器先于 domplate.js 解析时失败（IE8），可以在 onclick 前面加上 d- 前缀：

引用控制器 $this

在事件中，可以使用 $this 这个变量来获得当前控制器，可通过 $this 直接调用控制器上定义的方法和数据，对数据造成的更新，将直接反映到 html 视图上。

<div id="container">
  {{ count }}
  <button onclick=" ‌$this‍.data.count ++; ">Add</button>
  <button onclick=" ‌$this‍.subtract(); ">Subtract</button>
  <input type="number" value="{{ count }}" onkeyup=" ‌$this‍.data.count = this.value; ">
</div>
