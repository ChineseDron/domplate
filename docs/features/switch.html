Domplate 文档

switch-case-default

switch

用法：switch="expression"

switch 需和 case/default 属性配合使用，它所在的节点用于确定其影响的 case/default 的范围，expression 是一个标准的 JS 表达式，其值用于确定 case/default 节点的可见性，具体影响见下文。

<ul ‌switch‍="fruit.name">
  <li ‌case‍="apple">apple</li>
  <li ‌case‍="peach">peach</li>
  <li ‌case‍="pear">pear</li>
  <li ‌case‍="watermelon">watermelon</li>
  <li ‌default‍>other</li>
</ul>

case

用法：case="string"

case 属性所在的节点必须套于 switch 节点当中，当 swtich 表达式计算结果和 case 的值相等，则 case 所在的节点显示，否则隐藏。

需要注意的是：case 的值，在这里将用作字符串直接量，而不是 JS 表达式。
比如上述的 case="apple" 值是一个字符串 "apple"。

default

用法：default

default 属性所在的节点必须套于 switch 节点当中，当 swtich 表达式计算结果和所有的 case 的值都不相等，则 default 所在的节点显示，否则隐藏。

default 不需要值。

其它说明

  • case/default 均是可选节点，也可以有多个。
  • case/default 在计算可见性时，会向上遍历父节点，查找最近的 switch 节点作为参照，出于性能考虑，它们之间的父子关系会在初始化时直接确认好。
