Domplate 文档

数据初始化属性 init

数据在 domplate 模板里有两种初始化方法：

1）JS 中，使用 load 方法加载数据，详见 ctrl.load() 方法 一节；

2）HTML 中，使用 init 属性：

<div id="container" init="{ name: 'world' }">
  <p>Hello, {{ name }}!</p>
</div>

init 属性的值是一个标准的 JS 表达式，domplate 获取该表达式的值作为初始化值。

同时存在 init 属性和 load 方法时，以 load 方法为主。
