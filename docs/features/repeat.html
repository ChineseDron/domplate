Domplate 文档

repeat

用法：repeat="variable in expression"

repeat 属性的作用是使它所处的 dom 节点产生拷贝，拷贝次数取决于 expression 的成员个数，variable 是 expression 的成员，每一次拷贝时指向它的其中一个成员，variable 可以在该节点以及所有子节点中被访问.

索引值 $index

除了 variable 外，还可以使用 $index 这个固定名称的变量查询当前的索引值，expression 可以是数组或是对象，当它是数组时，$index 是其成员的顺序数字表示；当它是对象时，$index 是当前成员在对象中的键值，这点跟 JS 中的 for in 很象。

<ul>
  <li ‌repeat‍="‌todo‍ in ‌todos‍">
    <span>{{ ‌$index‍ }}</span>
    <input type="checkbox" d-checked="{{ ‌todo‍.done }}">
    <a>{{ ‌todo‍.content }}</a>
  </li>
</ul>
