Domplate 文档

show

用法：show="expression"

show 属性用于决定它所处的 dom 节点是否可见，其中值部分是一个标准的 JS 表达式，当该表达式的计算结果为 true 时，dom 节点可见，否则不可见。

<!-- 如果 count 的值大于 5，则 div 显示，否则隐藏 -->
<div ‌show‍="count > 5"></div>

关于隐藏

domplate 在隐藏某个 dom 节点时，会使用一个样式类 .domplate-hide，这个类的具体定义如下：

.domplate-hide{
  display: none !important;
}

这个样式类在页面加载 domplate.js 之后会被顺便添加到样式表当中，domplate 使用 important 来提升 display 的优先级。后面对于隐藏节点，也都是使用这个样式类。
